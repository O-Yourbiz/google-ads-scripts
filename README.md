# Google Ads Scripts Collection

Збірка корисних скриптів для автоматизації та моніторингу Google Ads.

---

## Список доступних скриптів:

### 1. [MCC Balance Monitor (Telegram Bot)](./mcc_balance_telegram_bot/)
Автоматична перевірка залишків бюджету на всіх акаунтах Керуючого акаунта (MCC) та надсилання сповіщень у Telegram при низькому балансі.
- **Можливості**: автоматичний розрахунок залишку з ПДВ, кастомні ліміти для кожного акаунта через Google Таблиці.
- **Детальніше**: див. [інструкцію в папці скрипта](./mcc_balance_telegram_bot/README.md).
